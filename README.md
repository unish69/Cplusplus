# 📘 2nd Semester C++ Programs

A collection of C++ programs written during my 2nd semester, covering fundamental concepts of programming in C++.

---

## 📂 Programs Included

| File | Description |
|------|-------------|
| `helloworld.cpp` | Prints "Hello World" to the screen, the classic first program |
| `printtheinputvalue.cpp` | Takes a number as input from the user and displays it |
| `averageoftwonumbers.cpp` | Takes two numbers as input and calculates their average |
| `FarenheitToCelcius.cpp` | Converts a temperature value from Fahrenheit to Celsius |
| `inline.cpp` | Uses an inline function to check whether a student has passed or failed based on marks |

---

## 🧠 Concepts Covered

- Basic input/output using `cin` and `cout`
- Variable declaration and data types (`int`, `float`, `string`)
- Arithmetic operations
- Conditional statements (`if-else`)
- Inline functions
- `getline()` for string input
- `endl` and `system("pause")`

---

## 🛠️ How to Compile & Run

Make sure you have a C++ compiler installed (e.g., **g++** via MinGW on Windows or GCC on Linux/macOS).

```bash
g++ helloworld.cpp -o helloworld
./helloworld
```

Replace `helloworld.cpp` with any program filename you want to run.

> **On Windows**, you can also open these files in **VS Code** and run them using the Code Runner extension or the built-in terminal.

---

## 📌 Notes

- These programs are beginner-level and written for learning purposes.
- `system("pause")` is used in some programs to keep the console window open on Windows. It can be removed on Linux/macOS.
- The `FarenheitToCelcius.cpp` file uses `int` for Fahrenheit input and `float` for the Celsius result to handle decimal precision.
